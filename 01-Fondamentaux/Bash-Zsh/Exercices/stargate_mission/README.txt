Bienvenue, recrue. Vous avez ete affecte(e) a l'equipe SG-nouveau.

CONTEXTE:
La cite d'Atlantis, dans la galaxie de Pegase, est en danger.
Ses reserves d'energie sont presque epuisees. Sans un ZPM
(Zero Point Module), la cite sombrera dans l'ocean.

VOTRE MISSION:
1. Infiltrer le SGC (Stargate Command) sous Cheyenne Mountain
2. Retrouver les coordonnees d'une planete Ancienne
3. Traverser la Porte des Etoiles
4. Recuperer un ZPM fonctionnel
5. Sauver Atlantis

COMPETENCES REQUISES:
- Navigation dans un systeme de fichiers (cd, ls)
- Lecture de documents (cat)
- Manipulation de fichiers (mkdir, touch, echo, mv)
- Recherche dans des bases de donnees (grep)
- Execution de scripts (chmod, bash)

POINT DE DEPART:
Rendez-vous au SGC. Entrez dans le dossier "sgc/" et commencez
par la surface de Cheyenne Mountain.

Bonne chance, soldat. Le sort d'Atlantis est entre vos mains.

- General Hank Landry, commandant du SGC
