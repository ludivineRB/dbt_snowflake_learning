## üìö Programme

### Lecon 1 - [Introduction aux Projets Data](01-introduction-projet-data.md)
- Qu'est-ce qu'un projet data ?
- Differences avec les projets logiciels classiques
- Cycle de vie d'un projet data
- Roles cles et matrice RACI
- Types de projets data et ecueils courants

### Lecon 2 - [Cadrage et Expression de Besoins](02-cadrage-expression-besoins.md)
- Phase de cadrage : la phase la plus critique
- Audit des besoins metiers et techniques d'entretien
- Cahier des charges fonctionnel
- User Stories et criteres d'acceptation
- Definition of Done pour les projets data

### Lecon 3 - [Methodes Agiles pour les Projets Data](03-methodes-agiles.md)
- Pourquoi l'Agile pour les projets data ?
- Framework Scrum adapte aux projets data
- Kanban pour la maintenance et le support
- Outils : Jira, Trello, GitHub Projects, Azure DevOps
- Comparaison Scrum vs Kanban

### Lecon 4 - [Objectifs SMART et Priorisation RICE](04-objectifs-smart-priorisation-rice.md)
- Methode SMART : definir des objectifs mesurables
- Framework RICE : prioriser les fonctionnalites
- Methode MoSCoW
- Combiner SMART + RICE pour une roadmap data

### Lecon 5 - [Faisabilite Technique et Financiere](05-faisabilite-technique-financiere.md)
- Faisabilite technique : audit des donnees, POC, Go/No-Go
- Faisabilite financiere : TCO, ROI, categories de couts
- Analyse des risques : SWOT, matrice de risques
- Exemples concrets : pipeline data, modele ML

### Lecon 6 - [Documentation et Communication](06-documentation-communication.md)
- Documentation technique : DAT, specifications, dictionnaire de donnees
- Communication avec le metier : traduction technique/business
- Reporting et gestion des parties prenantes
- Docs as Code : documentation versionnee

### Lecon 7 - [Accessibilite et Eco-conception](07-accessibilite-eco-conception.md)
- Accessibilite numerique : RGAA, WCAG 2.1
- Impact sur les produits data : dashboards, rapports, applications
- Eco-conception et Green IT
- Optimisation de l'empreinte carbone des traitements data
- Outils et bonnes pratiques

### [üìù Exercices Pratiques](08-exercices.md)
- Niveau 1 : Bases (User Stories, SMART, RACI)
- Niveau 2 : Intermediaire (RICE, cahier des charges, SWOT)

- Solutions detaillees
